... Conceptos aprendidos ...

1. Consultas SELECT avanzadas: Profundizar en el uso de la cláusula SELECT para realizar consultas más complejas, incluyendo la combinación de datos de múltiples tablas mediante JOIN.

2. Funciones de agregación: Aprender a utilizar funciones de agregación como COUNT, SUM, AVG, MIN y MAX para realizar cálculos sobre conjuntos de datos.

3. Grupos y agrupaciones: Comprender cómo agrupar datos utilizando la cláusula GROUP BY y filtrar resultados utilizando la cláusula HAVING.

4. Joins avanzados: Explorar diferentes tipos de JOIN (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN) y entender cuándo y cómo utilizarlos en consultas SQL.

5. Integridad referencial: Aprender sobre las relaciones entre tablas y cómo mantener la integridad referencial utilizando claves primarias y foráneas.
