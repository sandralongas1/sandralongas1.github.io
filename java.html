... Conceptos aprendidos ...

1. Programación orientada a objetos: Ampliar el conocimiento sobre este paradigma de programación, incluyendo conceptos como clases, objetos, herencia, polimorfismo, encapsulamiento, etc.

2. Algoritmos: Estudiar algoritmos más avanzados y eficientes, como algoritmos de búsqueda y ordenación (por ejemplo, búsqueda binaria, ordenación rápida), y cómo aplicarlos en la resolución de problemas.

3. Manejo de excepciones: Aprender a manejar errores y excepciones de manera más sofisticada para mejorar la robustez de los programas.

4. Pruebas y depuración: Aprender técnicas avanzadas de pruebas unitarias, integración continua y depuración de código para garantizar la calidad del software desarrollado.
