... Conceptos aprendidos ...

1. HTML Semántico: Comprender la importancia de etiquetas semánticas como <header>, <nav>, <main>, <footer>, etc., para mejorar la accesibilidad y la estructura del documento HTML.

2. Formularios HTML: Aprender a crear formularios usando etiquetas como <form>, <input>, <textarea>, <select>, etc., y entender cómo procesar la información enviada por el usuario.

3. CSS: Introducción al lenguaje de estilos CSS para dar estilo y diseño a las páginas HTML.

4. Responsive Design: Conceptos básicos de diseño web receptivo para que las páginas se vean bien en diferentes dispositivos y tamaños de pantalla.

5. Integración de multimedia: Incorporar imágenes, audio y video en las páginas web utilizando etiquetas HTML adecuadas.

6. Herramientas de desarrollo web: Familiarizarse con herramientas como editores de código, navegadores web y herramientas de inspección para desarrollar y depurar páginas web.

7. Prácticas de codificación limpia: Aprender buenas prácticas de codificación HTML para escribir código limpio, legible y mantenible.
